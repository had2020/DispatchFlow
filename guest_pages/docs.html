Docs

This documentation assumes that you wish to establish your own version of the website for your team of people.

This documentation also requires that you have some sort of computer and a route to run the server, so that your browser can connect to it. Hence why you wish to set up your node.

Server/Node Minimum requirements

Hardware: 64-bit CPU with at least One Gigabyte of Ram.

Software: Operating system with a package manager like Linux.

First Steps

Keep in mind sudo apt install will not work for each Linux distro. Ubuntu based such as the one's that get installed on a Rasiberry pi have apt package installer.

You can type commands in the termial in order to install the packages, which the server requires.

Type this command to install more recent syste packages, to uphold system cohesion.

sudo apt update

Next we create a folder for the server and then enter it.

mkdir dispatch_server

cd dispatch_server

Then we download the DispatchFlow server

wget https://github.com/had2020/DispatchFlow.git

Preping your OS

Installing Rust programming langauge to run the server.

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

source $HOME/.cargo/env

installing SQLlite

sudo apt install sqlite3

Running the server

ROCKET_ADDRESS=REPLACE_WITH_IPV4_ADDRESS ROCKET_PORT=REPLACE_WITH_OPENED_PORT cargo run

Note you must replace the text, REPLACE_WITH_IPV4_ADDRESS with your networks Ipv4 address and REPLACE_WITH_OPENED_PORT with the NAT fowarded port
